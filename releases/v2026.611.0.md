# Slaw v2026.611.0

First public npm release of Slaw under the `slaw-ai` organisation.

## Highlights

- `npx slaw onboard --yes` now works: the `slaw` CLI and all `@slaw/*` packages are published to npm.
- Package metadata now points at the public repository (github.com/slaw-ai/slaw).

## Notes

- Install: `npx slaw onboard --yes` then `npx slaw run`.
- See the [quickstart](https://docs.slaw.run) for LAN/tailnet binding options.
