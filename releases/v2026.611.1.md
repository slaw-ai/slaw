# Slaw v2026.611.1

First complete public npm release. Supersedes the partial v2026.611.0
(the unscoped CLI name `slaw` was rejected by the npm registry's
name-similarity policy; the CLI now publishes as `@slaw-ai/slaw`).

## Highlights

- Install with `npx @slaw-ai/slaw onboard --yes`, then `npx @slaw-ai/slaw run`.
- All runtime packages publish under the `@slaw-ai` scope.

## Notes

- The installed binary is still `slaw` (e.g. after `npm i -g @slaw-ai/slaw`).
- See the [quickstart](https://docs.slaw.run) for LAN/tailnet binding options.
